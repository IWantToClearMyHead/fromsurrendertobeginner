### 函数
```
int *p[n]
int (*p)[n]
int (*p)(n1, n2)
int *p(n1, n2)
```
前两个写法理清之后可以到后两个，至此回的四种写法终于结束。
究竟什么是函数？
数学上函数很简单，`y = f(x)`.
代码里也是如此，通常有一个定义，然后就是调用。学术名词在参数上又多了两个，一为形式，二为实际。
